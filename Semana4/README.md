# Semana 4 — Objetivos y tareas

Plantilla para la Semana 4. Rellena con las actividades, recursos y entregables específicos.

## Resumen

Objetivo de la semana: estabilizar la aplicación, optimizar rendimiento y preparar revisión intermedia.

## Actividades sugeridas

1. Optimizar consultas y operaciones críticas.
2. Realizar pruebas de carga básicas y documentar resultados.
3. Preparar demo o presentación con los avances.

## Entregables

- Informe breve de rendimiento.
- Lista de optimizaciones aplicadas.
- Material para demo (capturas, scripts).

## Checklist

- [ ] Pruebas de carga ejecutadas
- [ ] Informe de rendimiento
- [ ] Demo preparada
