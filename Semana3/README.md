# Semana 3 — Objetivos y tareas

Plantilla para la Semana 3. Rellena con las actividades, recursos y entregables específicos.

## Resumen

Objetivo de la semana: consolidar integraciones, añadir monitorización básica y preparar despliegue local.

## Actividades sugeridas

1. Integrar servicios dependientes (por ejemplo, base de datos, broker de mensajes).
2. Añadir logs y métricas básicas para observabilidad.
3. Probar escenarios de fallo y documentar resultados.

## Entregables

- Documentación de integración y despliegue.
- Scripts o docker-compose para levantar dependencias (si aplica).

## Checklist

- [ ] Integraciones probadas
- [ ] Logs/metrics activados
- [ ] Documentación de despliegue lista
